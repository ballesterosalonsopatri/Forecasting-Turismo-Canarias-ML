# Forecasting-Econ-mico-del-Sector-Tur-stico-Canario-mediante-T-cnicas-de-Machine-Learning
Proyecto de Machine Learning orientado al forecasting del sector turístico canario. A partir de datos del ISTAC (2018–2024), se desarrollarán modelos predictivos que analicen actividades, territorios y tipos de estancia para estimar la demanda turística futura y su impacto económico.
